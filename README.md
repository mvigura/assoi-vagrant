# ASSOI Compose

## Prerequisites
- git
- [Vagrant](https://www.vagrantup.com/downloads.html)
- [VirtualBox](https://www.virtualbox.org)

## Installation
1. ```cd /path/to/working/directory```
1. ```git clone --recursive git@github.com:mvigura/assoi-vagrant.git```


## Up
1. ```cd /path/to/assoi-vagrant```
1. ```vagrant up```

## Other
- ```vagrant ssh```
- ```vagrant destroy```
